# DataxLabs-Internship-Task-5

# Titanic Dataset EDA

## Objective
Perform Exploratory Data Analysis (EDA) on the Titanic dataset to identify trends, patterns, and relationships.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data Inspection
- Missing Value Analysis
- Histograms
- Boxplots
- Scatterplots
- Heatmap
- Pairplot

## Key Findings
- Females survived more than males.
- First-class passengers had better survival rates.
- Fare positively influenced survival.
- Significant missing values exist in Age and Cabin.

## Conclusion
EDA revealed important factors influencing survival and identified data quality issues that require preprocessing.

Important Findings for Report
Finding 1

549 passengers did not survive while 342 survived.

Finding 2

Female passengers had substantially higher survival rates.

Finding 3

Passengers traveling in First Class had better chances of survival.

Finding 4

Fare distribution is highly right-skewed with many outliers.

Finding 5

Age has missing values and requires preprocessing before modeling.

Finding 6

Cabin has over 75% missing values.

Finding 7

Fare and Passenger Class are important factors affecting survival.

Summary of Findings
The dataset contains 891 passenger records.
Missing values are present in Age, Cabin, and Embarked.
Survival rate is approximately 38%.
Females were more likely to survive than males.
First-class passengers had the highest survival probability.
Fare is positively associated with survival.
Several outliers exist in Fare.
Cabin contains excessive missing values and may need removal or feature engineering.
